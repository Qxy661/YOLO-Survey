## 1. 项目简介

本项目对 **YOLO 系列目标检测算法** 及其相关技术进行系统性调研，覆盖从基础概念到前沿研究的完整知识体系。项目以文档为核心，辅以思维导图和参考文献，力求为读者提供一份**可查阅、可引用、可复现**的技术资料库。

### 核心内容

| 维度 | 说明 |
|------|------|
| 算法演进 | YOLOv1 到 YOLOv8/v9/v10 的版本变迁与技术突破 |
| 专题深入 | 小目标检测、损失函数设计、推理优化等关键方向 |
| 论文导读 | 精选论文的核心思想、方法与个人理解 |
| 实验实践 | 复现指南、消融实验设计与结果分析 |

---

## 2. 适合人群

本项目适合以下读者：

- **有一定深度学习基础**，了解 CNN、backbone、anchor 等基本概念
- **想系统了解 YOLO 系列**的发展脉络和技术细节
- **从事目标检测相关工作**（科研/工程/竞赛），需要快速查阅参考资料
- **正在准备论文或报告**，需要综述性质的材料

> 如果你是完全零基础的深度学习入门者，建议先学习 PyTorch 基础和 CNN 基础，再回来阅读本项目。

## 3. 学习路线推荐

根据不同目标和时间预算，我们推荐三条学习路线：

### 3.1 入门路线（约 2 天）

> **目标**：快速建立对 YOLO 和目标检测的整体认知。

| 序号 | 阅读内容 | 预计时间 | 说明 |
|:----:|----------|:--------:|------|
| 1 | [01-目标检测概述](01-基础概念/01-目标检测概述.md) | 1h | 了解目标检测的定义、任务类型和经典方法 |
| 2 | [02-评估指标体系](01-基础概念/02-评估指标体系.md) | 1h | 掌握 mAP、IoU 等核心评估指标 |
| 3 | [03-损失函数](01-基础概念/03-损失函数.md) | 1h | 理解检测任务中的损失函数设计思路 |
| 4 | [02-YOLO演进专题 (前3篇)](02-YOLO演进专题/) | 2h | 了解 YOLOv1 到 YOLOv3 的演进 |
| 5 | [06-YOLO系列全景对比](02-YOLO演进专题/06-YOLO系列全景对比.md) | 1h | 从全局视角对比不同版本的特点 |

**完成后你应该能够**：
- 用自己的话解释 YOLO 是什么
- 理解 mAP 的含义和计算方式
- 区分 YOLOv1/v2/v3 的核心改进

> **目标**：深入理解 YOLO 的关键技术细节和优化方向。

| 序号 | 阅读内容 | 预计时间 | 说明 |

## 5. 如何使用本项目

### 5.1 在线阅读

所有文档均为 Markdown 格式，可直接在 GitHub 上浏览。文档间的相对链接已做好配置，点击即可跳转。

### 5.2 本地阅读

```bash
# 克隆项目
git clone <repo-url>
cd YOLO-Survey

# 使用你喜欢的 Markdown 阅读器打开
# 推荐：VS Code + Markdown Preview Enhanced 插件
code docs/00-导读与学习路线.md
```

### 5.3 文档结构约定

- 每篇文档开头标注**预计阅读时间**和**难度等级**
- 文档内使用 `> 引用块` 标注**个人理解**和**关键提示**
- 使用 `表格` 进行**横向对比**，使用 `代码块` 展示**公式或伪代码**
- 文档末尾附**参考文献**和**延伸阅读**链接

### 5.4 符号说明

| 符号 | 含义 |
|------|------|
| :star: | 重点/核心知识点 |
| :bulb: | 个人理解和提示 |
| :warning: | 易错点或注意事项 |
| :link: | 相关文档链接 |
| :book: | 原始论文引用 |

### 5.5 反馈与贡献

- 发现错误或有改进建议，欢迎提 Issue
- 希望补充某个专题，欢迎提 Feature Request
- 直接提交 Pull Request 也是欢迎的
