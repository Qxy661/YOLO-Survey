### 2.2 标签格式转换

```python
# convert_visdrone_to_yolo.py
import os
from pathlib import Path

# VisDrone 类别映射 (排除 ignored regions)
VISDRONE_CLASSES = {
    0: "pedestrian",
    1: "people",
    2: "bicycle",
    3: "car",
    4: "van",
    5: "truck",
    6: "tricycle",
    7: "awning-tricycle",
    8: "bus",
    9: "motor",
}

def convert_annotation(src_path, dst_path, img_w, img_h):
    """将 VisDrone 格式转为 YOLO 格式"""
    with open(src_path, 'r') as f:
        lines = f.readlines()

    yolo_lines = []
    for line in lines:
        parts = line.strip().split(',')
        x, y, w, h = int(parts[0]), int(parts[1]), int(parts[2]), int(parts[3])
        cls_id = int(parts[5])

        if cls_id < 0 or cls_id > 9:  # 忽略 ignored/other
            continue

        # 转换为中心点 + 宽高, 归一化
        cx = (x + w / 2) / img_w
        cy = (y + h / 2) / img_h
        nw = w / img_w
        nh = h / img_h

        yolo_lines.append(f"{cls_id} {cx:.6f} {cy:.6f} {nw:.6f} {nh:.6f}")

    with open(dst_path, 'w') as f:
        f.write('\n'.join(yolo_lines))
```

## 3. 训练脚本

### 3.1 基础训练

```bash
# 使用默认配置训练 YOLOv8n
yolo detect train \
  model=yolov8n.pt \
  data=visdrone.yaml \
  epochs=100 \
  imgsz=640 \
  batch=16 \
  device=0 \
  project=runs/detect \
  name=baseline
```

### 3.2 Python API 训练

```python
from ultralytics import YOLO

model = YOLO("yolov8n.pt")

results = model.train(
    data="visdrone.yaml",
    epochs=100,
    imgsz=640,
    batch=16,
    device=0,
    project="runs/detect",
    name="baseline",
    # 数据增强参数
    hsv_h=0.015,
    hsv_s=0.7,
    hsv_v=0.4,
    degrees=0.0,
    translate=0.1,
    scale=0.5,
    fliplr=0.5,
    mosaic=1.0,
    mixup=0.0,
)
```

### 3.3 训练超参数参考

| 参数 | 基线值 | 我们使用的值 | 说明 |
|------|-------|------------|------|
| epochs | 100 | 100 | 训练轮数 |
| batch | 16 | 16 | 批大小 (受 VRAM 限制) |
| imgsz | 640 | 640 / 1024 | 输入尺寸 |
| lr0 | 0.01 | 0.01 | 初始学习率 |
| lrf | 0.01 | 0.01 | 最终学习率 = lr0 * lrf |
| momentum | 0.937 | 0.937 | SGD 动量 |
| weight_decay | 0.0005 | 0.0005 | 权重衰减 |
| warmup_epochs | 3.0 | 3.0 | 预热轮数 |

## 4. 评估流程

### 4.1 标准评估

```bash
# 在验证集上评估
yolo detect val \
  model=runs/detect/baseline/weights/best.pt \
  data=visdrone.yaml \
  imgsz=640 \
  batch=16 \
  device=0
```

### 4.2 评估指标说明

| 指标 | 含义 | 我们重点关注 |
|------|------|------------|
| mAP@0.5 | IoU=0.5 时的 mAP | 主要对比指标 |
| mAP@0.5:0.95 | IoU 0.5~0.95 平均 | 综合评估 |
| Precision | 精确率 | 误检情况 |
| Recall | 召回率 | 漏检情况 |
| AP per class | 每类 AP | 分析弱项类别 |

### 4.3 单张图片推理

```python
from ultralytics import YOLO

model = YOLO("runs/detect/baseline/weights/best.pt")

# 推理并可视化
results = model.predict(
    source="test_image.jpg",
    conf=0.25,
    iou=0.45,
    save=True,
    project="runs/detect",
    name="predict",
)

# 获取结果
for r in results:
    print(f"Detected {len(r.boxes)} objects")
    for box in r.boxes:
        cls = int(box.cls[0])
        conf = float(box.conf[0])
        xyxy = box.xyxy[0].tolist()
        print(f"  Class {cls}: {conf:.2f} at {xyxy}")
```

### 5.2 SAHI 推理脚本

```python
from sahi import AutoDetectionModel
from sahi.predict import get_sliced_prediction

# 加载模型
detection_model = AutoDetectionModel.from_pretrained(
    model_type="ultralytics",
    model_path="runs/detect/baseline/weights/best.pt",
    confidence_threshold=0.25,
    device="cuda:0",
)

# SAHI 切片推理
result = get_sliced_prediction(
    image="test_image.jpg",
    detection_model=detection_model,
    slice_height=640,
    slice_width=640,
    overlap_height_ratio=0.2,
    overlap_width_ratio=0.2,
)

# 获取结果
print(f"Detected {len(result.object_prediction_list)} objects")
for pred in result.object_prediction_list:
    print(f"  {pred.category.name}: {pred.score.value:.2f}")
```

## 6. 自定义模块注册

### 6.1 register_custom_modules.py

当修改了 Ultralytics 的源码（如添加新的损失函数、新的网络模块），需要在入口处注册：

```python
# register_custom_modules.py
"""
在训练/推理脚本开头导入此文件，注册自定义模块。
"""

import ultralytics.nn.modules as modules
from ultralytics.nn.tasks import DetectionModel

# 示例: 注册自定义注意力模块
class CBAM(modules.Module):
    """CBAM: Convolutional Block Attention Module"""
    def __init__(self, c1, reduction=16, kernel_size=7):
        super().__init__()
        # ... 实现 ...

    def forward(self, x):
        # ... 实现 ...
        pass

# 将自定义模块注册到全局命名空间
modules.CBAM = CBAM

# 如果修改了 DetectionModel 的解析逻辑
# 需要修改 ultralytics/nn/tasks.py 中的 parse_model 函数
```

### 6.2 使用自定义模块的 YAML 配置

```yaml
# yolov8n-cbam.yaml
# 在 backbone 或 head 中使用自定义模块
backbone:
  - [-1, 1, Conv, [64, 3, 2]]     # 0-P1/2
  - [-1, 1, Conv, [128, 3, 2]]    # 1-P2/4
  - [-1, 3, C2f, [128, True]]
  - [-1, 1, Conv, [256, 3, 2]]    # 3-P3/8
  - [-1, 6, C2f, [256, True]]
  - [-1, 1, CBAM, [256]]          # 自定义 CBAM 模块
  # ... 其余层
```

### 7.3 自定义日志记录

```python
import json
from pathlib import Path

def log_experiment(exp_name, params, results):
    """记录实验参数和结果到 JSON"""
    log_file = Path("experiments_log.json")

    log = {}
    if log_file.exists():
        with open(log_file) as f:
            log = json.load(f)

    log[exp_name] = {
        "params": params,
        "results": results,
    }

    with open(log_file, 'w') as f:
        json.dump(log, f, indent=2)

# 使用示例
log_experiment(
    exp_name="round1_baseline",
    params={"model": "yolov8n", "imgsz": 640, "epochs": 100},
    results={"mAP50": 0.2979, "mAP50-95": 0.1612}
)
```

## 8. 常见问题与解决方案

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| CUDA out of memory | batch size 过大 | 减小 batch size 或 imgsz |
| 训练 loss 为 NaN | 学习率过大 | 降低 lr0, 检查数据标签 |
| mAP 很低但 loss 下降 | 过拟合或数据问题 | 检查标签正确性, 增加数据增强 |
| 导出 ONNX 报错 | 自定义算子不支持 | 注册自定义算子或替换实现 |
| TensorRT 版本不匹配 | CUDA / TensorRT 版本冲突 | 使用匹配的版本组合 |
| SAHI 推理很慢 | 切片过多 | 增大切片尺寸或重叠比例 |
| 多 GPU 训练报错 | DDP 配置问题 | 使用 `device=0` 单卡训练 |
